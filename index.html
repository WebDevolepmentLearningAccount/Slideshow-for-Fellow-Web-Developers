HTML

This is the first language to learn while doing web developement and the most basic language in the series of web developement. It stands for Hyper Text Markup Language.

CSS

CSS is the second language you wil learn in web developement. It is used to style HTML. It stands for Cascading Style Sheets.

JS

JS is the last language that you wil learn in web developement. Its used to add life to your pages and make changes on the spot in your html/css. It stands for JavaScript.
